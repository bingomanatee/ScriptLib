Alerts

Demonstration

Success     Notice     Warning     Error     Success sticky     Notice sticky     Warning sticky     Error sticky

Explanation

Success

Not sticky(default):
function showSuccessToast() {
$().toastmessage('showSuccessToast', "Success Dialog which is fading away ...");
}
Sticky:
function showStickySuccessToast() {
$().toastmessage('showToast', {
text : 'Success Dialog which is sticky',
sticky : true,
position : 'top-right',
type : 'success',
closeText: '',
close : function () {
console.log("toast is closed ...");
}
});
} Show:
<a href="javascript:showSuccessToast();">not sticky</a>
<a href="javascript:showStickySuccessToast();">sticky</a> not sticky     sticky
